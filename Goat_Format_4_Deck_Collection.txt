- Shining Angel gets LIGHT monsters (Magician of Faith, D.D. Warrior Lady)
- Giant Rat gets Earth monsters (Exiled Force)

================================
MATCHUP GUIDE
================================

GOAT CONTROL vs GOAT BEATDOWN: 55-45 (Control favored)
- Control has more answers and card advantage
- Beatdown can win if aggressive early

GOAT CONTROL vs CHAOS RECRUITER: 50-50 (Even)
- Both play similar grind game
- Recruiter is faster, Control has more answers

GOAT CONTROL vs WARRIOR TOOLBOX: 45-55 (Toolbox favored)
- Toolbox can adapt to Control's strategy
- Multiple ROTA provides consistent answers

GOAT BEATDOWN vs CHAOS RECRUITER: 55-45 (Beatdown favored)
- Beatdown's aggression can overwhelm before Chaos setup
- Recruiter can stabilize if they survive early game

GOAT BEATDOWN vs WARRIOR TOOLBOX: 50-50 (Even)
- Both are aggressive strategies
- Toolbox has better answers, Beatdown has bigger monsters

CHAOS RECRUITER vs WARRIOR TOOLBOX: 50-50 (Even)
- Recruiter has bigger finishers
- Toolbox has more consistent answers

================================
DIFFICULTY RANKING
================================

EASIEST → HARDEST:
1. Goat Beatdown (straightforward aggression)
2. Chaos Recruiter (simple combo setup)
3. Warrior Toolbox (requires knowing matchups)
4. Goat Control (requires reading opponent and timing)

================================
TEACHING RECOMMENDATIONS
================================

START HERE: Goat Beatdown
- Simplest strategy: play monsters, attack
- Teaches basic resource management
- Minimal complex decisions

NEXT: Chaos Recruiter
- Introduces graveyard setup
- Teaches counting resources (LIGHT/DARK)
- Clear combo execution

THEN: Warrior Toolbox
- Teaches adaptation and matchup knowledge
- Introduces toolbox concepts
- Requires understanding what to search when

FINALLY: Goat Control
- Most complex decision trees
- Teaches when to use removal vs when to wait
- Requires understanding all matchups
